# Node Banana Documentation

Welcome to the official documentation for **Node Banana** — a visual workflow editor for AI image generation.

## What is Node Banana?

Node Banana is a node-based application that lets you create complex AI image generation workflows by connecting nodes on an infinite canvas. Think of it as a visual programming environment where each node performs a specific task, and you wire them together to build powerful image pipelines.

## Key Features

- **Visual Node Editor** — Drag, drop, and connect nodes on an infinite canvas with pan, zoom, and multi-select
- **Multi-Provider AI Generation** — Generate images and videos using Gemini, Replicate, or fal.ai with unified model discovery
- **Video Generation** — Create AI-generated videos with dedicated video nodes and playback support
- **AI Text Generation** — Create prompts using Gemini or OpenAI language models
- **Image Annotation** — Draw on images with shapes, arrows, and text using a full-featured editor
- **Workflow Templates** — Get started quickly with 6 preset templates or generate custom workflows with AI
- **Cost Tracking** — Monitor your API usage costs in real-time
- **Save & Load** — Export workflows as JSON files with automatic saving
